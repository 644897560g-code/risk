# 印尼现金贷风控特征文档

## 一、数据源概述

### 1.1 申请信息 (base)
| 字段 | 类型 | 说明 |
|------|------|------|
| appname | string | 产品线名称 (flex-rupiah / teman-pinjam) |
| birthday | date | 客户生日 (DD-MM-YYYY) |
| job | string | 职业代码 |
| gender | int | 性别 (0=女, 1=男) |
| salary | int | 月收入 (IDR) |
| workYears | int | 工作年限 |
| marita | int | 婚姻状况 |
| children | int | 子女数量 |

### 1.2 应用列表 (applist)
每个客户手机安装的应用列表，包含：
- **packageName**: 包名（唯一标识）
- **appName**: 应用名称
- **installTime**: 安装时间（时间戳）
- **updateTime**: 更新时间

### 1.3 FDC信用报告
基于印尼OJK三方征信数据：
- **history_inquiry**: 历史查询记录（机构名、查询时间、查询原因）
- **pinjaman**: 贷款记录（机构、金额、状态、放款/还款日期）
- **platform_aktif**: 当前活跃平台

---

## 二、特征设计原则

1. **防穿越**：特征计算只使用申请时间之前的数据
2. **业务可解释**：每个特征需有明确的业务含义
3. **统计显著性**：IV >= 0.02，PSI <= 0.25，覆盖率 > 5%
4. **首贷聚焦**：针对首次贷款客户的特征挖掘

---

## 三、特征类别

### 3.1 FDC衍生特征
- 近3/7/30天查询机构数量
- 现有贷款笔数及总金额
- 历史逾期次数及逾期率
- 近N天新增机构数

### 3.2 APP安装特征
- 高风险APP安装数（赌博、借贷类）
- 金融类APP占比
- APP安装/更新时间特征（RFM）

### 3.3 交叉特征
- 收入与FDC债务比
- 年龄与APP类别偏好
- 区域与信贷历史关联
